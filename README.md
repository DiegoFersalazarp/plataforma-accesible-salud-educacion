# Plataforma Web Accesible – Salud y Educación

Proyecto académico correspondiente a la Fase 4 del curso, cuyo objetivo es el desarrollo de un prototipo funcional (MVP) de una plataforma web accesible que facilite el acceso a servicios de salud y educación para personas con discapacidad en la ciudad de Bogotá.

## Metodología
El desarrollo del prototipo se realizó aplicando la metodología ágil Scrum, organizando el trabajo en sprints orientados a la entrega incremental de funcionalidades.

## Alcance del repositorio
Este repositorio contiene:
- Documentación del proyecto
- Estructura base del prototipo
- Evidencia del proceso de desarrollo académico

## Funcionalidades principales
- Prototipo navegable
- Flujos de interacción definidos
- Operaciones CRUD (crear, consultar, actualizar y eliminar)
- Cumplimiento de criterios de accesibilidad WCAG 2.1

## Nota
Este repositorio corresponde a un ejercicio académico y no representa un sistema en producción.
